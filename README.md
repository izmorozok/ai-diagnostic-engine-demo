# AI Diagnostic Engine Demo

Demo architecture for AI-assisted diagnostic quizzes, JSON schemas, scoring logic, and portfolio-safe report examples.

## Purpose

This repository demonstrates how a diagnostic system can be structured:

- quiz structure
- question logic
- answer mapping
- scoring model
- result types
- report generation flow

## Public demo only

This is not a production diagnostic engine.

Commercial materials, full scoring rules, internal prompts, and production algorithms are not included.

## Русская версия

Этот репозиторий показывает демо-архитектуру AI-диагностики: структуру квиза, логику вопросов, связь ответов с механизмами, модель оценки и пример формирования результата.

Это публичная демо-версия для портфолио. Полная рабочая логика, коммерческие материалы и внутренние промпты не публикуются.
